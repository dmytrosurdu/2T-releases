# UOM

This folder contains the Universe-Originating Measurement manuscript and note stack.

## Contents

- `summary/summary.tex`: short summary. Read it first.
- `main.tex`: broad UOM manuscript.
- `2T_Main_interface.tex`: interface note to the Two Time Main Theorem lane.

- `.aux` files: cross-reference support files for linked manuscripts.
- `pdf/`: generated PDFs for standalone LaTeX documents in this folder.
- `summary/pdf/`: generated PDF for the summary manuscript.
