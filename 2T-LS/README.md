# 2T-LS

This folder contains the Two Time Limited Scope manuscript package.

## Contents

- `limited_scope_theorem_PRL_Letter.tex`: PRL Letter main manuscript.
- `limited_scope_theorem_translation.tex`: expanded translation-style manuscript. PRL Letter SM.
- `limited_scope_theorem_translation_clean.tex`: cleaned translation variant.
- `prl_macros.tex`: shared notation and manuscript macros.
- `pdf/`: generated PDFs for the standalone LaTeX documents in this folder.
