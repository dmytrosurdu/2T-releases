# 2T-Main

This folder contains the Two Time Main Theorem manuscript stack.

## Contents

- `main_theorem.tex`: PRD Article of Two Time Main Theorem.
- `appendix_main.tex`, `appendix_subprograms_4a-4b.tex`, `appendix_subprograms_4c_discussion.tex`: detailed proof.
- `macros_2t.tex`, `macros_ls.tex`: shared notation and formatting macros.
- `refs.bib`: bibliography database used by the manuscript.
- `main_theorem.aux`: cross-reference support file for linked manuscripts.
- `Mathematica/`: audit material, reports, notebooks, kernel code, and certificates.
- `pdf/`: generated PDF output for the standalone manuscript in this folder.
