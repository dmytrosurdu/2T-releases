# 2T Releases

This directory is the staged release bundle for the 2T workspace. It collects the manuscripts of both 2T theorems and Universe-Originating Measurement theory and notes, 2T-LS Agda formal proof, UOM simulators, and supporting UOM->Main audit lanes.

## Top-level folders

- `2T-LS/`: Two Time Limited Scope manuscript package. Standalone PDFs are in `2T-LS/pdf/`.
- `2T-Main/`: Two Time Main Theorem manuscript stack. It includes the Mathematica audit lane (mostly of UOM implementation of its interface) and the generated `2T-Main/pdf/main_theorem.pdf`.
- `TwoTimes/`: Agda formal proof of 2T-LS.
- `UOM/`: Universe-Originating Measurement manuscript and note stack, including generated PDFs in `UOM/pdf/` and the summary PDF in `UOM/summary/pdf/`.
- `uom_simulator_v1/`: baseline simulator release copy.
- `uom_simulator_v2/`: expanded simulator release copy with only the final retained combined multi-aeon output.
- `CTFM/`: additional optional textbook referenced in UOM manuscript for interested reader.
